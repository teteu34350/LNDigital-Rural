# Sistema de Gestão de Propriedades Rurais

Este projeto consiste no desenvolvimento de um sistema de gestão de propriedades rurais, criado com fins acadêmicos. O objetivo é aplicar conceitos de desenvolvimento web na criação de uma aplicação funcional voltada para a organização e controle de informações do ambiente rural.

O sistema foi desenvolvido utilizando o framework Django, permitindo a prática de conceitos como estruturação de projetos, manipulação de dados, criação de interfaces e organização de regras de negócio em uma aplicação web.

## Objetivo

Desenvolver uma aplicação web para gestão de propriedades rurais, com foco no aprendizado de desenvolvimento com Django, modelagem de dados e organização de sistemas, aplicando conceitos de programação em um contexto prático.

## Tecnologias Utilizadas

* Python
* Django
* SQLite (db.sqlite3)
* HTML
* CSS
